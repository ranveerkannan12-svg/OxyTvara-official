# OxyTvara — Revise Smarter

OxyTvara is a simple, fast, and clean study tool designed to help students revise more effectively.  
The goal is to make revision easier, more organised, and more enjoyable.

## 🚀 Live Website
The project is deployed on Netlify:  
https://oxytvara.netlify.app

## 📁 Project Structure
This repository contains the static files used to run the live site:
- index.html
- manifest.json
- sw.js
- /icons

These files are automatically deployed to Netlify on every push to the `main` branch.

## 🎯 Features (Current)
- Clean landing page with OxyTvara branding
- Create Account form (name, email, password)
- Avatar selection UI
- Mobile‑friendly layout

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Netlify (Hosting)
- GitHub (Version control)

## 📦 Deployment
This project uses **Netlify Continuous Deployment**.  
Any changes pushed to this repo will automatically redeploy the live site.
